# Demo assets

Put screenshots or GIFs here, then keep the README links in sync.

## Suggested captures

| File | What to show |
|------|----------------|
| `login.png` (or `.gif`) | SPA login page with demo hint visible |
| `tasks-list.png` | Authenticated task list (seeded + one create/edit if useful) |
| `swagger-auth.png` | Swagger with Authorize + successful `GET /api/tasks` (optional) |

## Tips

- Prefer PNG for stills; GIF/WebM for a short CRUD loop.
- Keep files reasonably small for GitHub (under ~2–5 MB each when possible).
- After adding files: `git add docs/assets && git commit` (or open a tiny docs PR).
